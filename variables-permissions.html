Variables & Permissions

A reference guide to all available variables and permission nodes used across Bliss-Bot.

Variables

Variables are dynamic placeholders used in system messages, auto responders, and embeds. They are replaced with live values when the message is sent.

User Variables

VariableOutput
{user}Member display name
{user.mention}Member mention ping
{user.id}Member user ID
{user.tag}Username#discriminator
{user.avatar}Avatar URL
{user.joinedAt}Server join date
{user.createdAt}Account creation date

Server Variables

VariableOutput
{server}Server name
{server.id}Server ID
{server.members}Total member count
{server.icon}Server icon URL
{server.boosts}Current boost count
{channel}Current channel name
{channel.mention}Channel mention link

Permissions

The following Discord permission names are recognized by Bliss-Bot's fake permissions and command restriction systems.

Permission NodeDescription
administratorFull server administrator access
manage_serverManage server settings
manage_channelsCreate, edit, delete channels
manage_rolesCreate, edit, delete roles
manage_messagesDelete and pin messages
ban_membersBan and unban members
kick_membersKick members
moderate_membersTimeout members
manage_webhooksCreate and manage webhooks
manage_nicknamesChange other members' nicknames
